Terrafy

A 2D sandbox survival game where the player has to gather resources and fend off enemy monsters. It has Multiplayer functionality, so you can play with your friends. Below are the repositories for the client and server, a report that I wrote for my coursework and a video testing it/showing off gameplay.

Project Overview

This game was made for my A-Level coursework in Computer Science and received 75/75. It was my first exposure to a larger project and the complexities that come along with it, such as version control. The objective of the game is for the player to kill 50 zombies, which spawn randomly and to kill the boss which spawns after the kill criteria has been met.

Terrafy was made in Java, using JavaFX as the GUI library. It procedurally generates each map using Midpoint Displacement, so each world is unique. The game also has Multiplayer functionality, so multiple users can play on the same world, building and crafting using resources gathered.

The server is written in Java, and uses Virtual Threading, so it can support many players at once. It assigns a new player to a new thread and keeps track of their output streams to check for new data.
