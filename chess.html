Chess

A game that allows two users to play chess locally. Has working checkmate, castling and en passant.

Project Overview

This was a game I made in my spare time to improve my programming skills. It allows 2 people to play chess against each other until someone checkmates a King. I focused on improving my understanding of Object-Oriented Programming techniques, which this project was perfect for, as the roles are very defined.

I used Java to make this game, with JavaFX as the GUI library. This project made me understand the importance of reaching out to more experienced developers, as had a recursive issue when putting the King into check that I couldn't solve after many attempts.

Feel free to check out the code through the links at the top of the page.
